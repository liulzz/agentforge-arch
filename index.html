总体架构概览
AgentForge 采用分层架构设计，从接入层到基础设施层提供完整的智能体全生命周期管理能力。支持 SaaS 多租户和子产品独立部署两种模式。
部署模式对比

☁️ SaaS 模式

  • 多租户共享基础设施，通过 Namespace 隔离
  • 统一 LLM Gateway，租户级速率限制与用量计费
  • 共享 Vector DB，按租户 namespace 隔离数据
  • CDN 加速静态资源，全球负载均衡
  • 自动弹性伸缩，按需付费
  • 统一升级，零停机更新
  • 适用场景：中小企业、快速接入、成本敏感

🏢 子产品模式

  • 单租户独立 K8S 集群，可私有化部署
  • 嵌入式 LLM Gateway，支持本地模型推理
  • 独立 DB / Cache / Storage，数据完全隔离
  • Helm Chart 一键安装，支持离线部署
  • 支持白标定制，品牌化输出
  • 独立版本管理，可定制化开发
  • 适用场景：大型企业、金融/政务、数据安全要求高
